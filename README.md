# Laravel VueJs Maintenance System

A simple Maintenance System using Laravel and VueJs

## Installation

```bash
git clone https://github.com/crsdngnn/maintenance.git

composer install

npm install

php artisan migrate -seed

Credentials:
Email: admin@admin.com
password: admin
```

## Run in Vuejs
```bash
npm run watch
```
## Run in Laravel
```bash
php artisan serve
```
